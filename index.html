System Overview

Connecting Data to Sustainable Action

A clean three-stage pipeline — collect, process, act.

1) Data Acquisition

IoT meters & sensors streaming solar, wind, EV, and grid readings from distributed nodes.

2) Central Processing

Spring Boot services persist to MySQL, validate with Jakarta Validation, secure with JWT.

3) Intelligent Insights

Web dashboard visualizes live consumption, cost, and mix with smooth real-time motion.

Data Flow
  • Client sends auth → /api/auth/login → gets JWT
  • Sensor fetch → /api/sensors/recent (user-scoped)
  • Summary → /api/user-energy/average
  • User config → /api/user-energy/set-cost
  • Export CSV → /api/user-energy/export
  • Simulator admin → /api/simulator/*
